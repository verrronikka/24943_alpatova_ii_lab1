## Задача 1

- Тип задачи: Классификация
- Датасет: The Simpsons Characters Data
- Метрика качества: Accuracy

Краткое описание задачи
Задача заключается в разработке модели машинного обучения для классификации персонажей мультсериала «Симпсоны» на основе их визуальных характеристик из набора данных The Simpsons Characters Data. Целью является автоматическое определение имени персонажа по изображению с максимизацией метрики Accuracy.

## Подготовка датасета

- Скачать zip-архив по ссылке: https://www.kaggle.com/datasets/alexattia/the-simpsons-characters-dataset
- Расархировать в папку data.
- Удалить из папки simpsons_dataset папку simpsons_dataset

## Как воспроизвести

Команда обучения:
```
python train.py
```
Команда валидации:
```
python val.py
```
